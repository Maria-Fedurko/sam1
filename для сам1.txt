1. По каналу связи передаются сообщения, содержащие только шесть букв: А, Б, В, Д, О, Т. Для передачи используется двоичный код, удовлетворяющий условию Фано. Кодовые слова для некоторых букв известны: Б – 010, В – 011. Какое наименьшее количество двоичных знаков потребуется для кодирования слова ВОДООТВОД? 

2. Для кодирования некоторой последовательности, состоящей из букв А, Б, В, Г, Д, решили использовать неравномерный двоичный код, удовлетворяющий условию Фано. Для букв А, Б, В, Г использовали соответственно кодовые слова 011, 010, 001, 0001. Укажите возможное кодовое слово для буквы Д, при котором код будет допускать однозначное декодирование. Если таких кодов несколько, укажите код с наименьшим числовым значением.
